# Admin Analytics Dashboard

## Tech Stack

* React (Frontend)
* Node.js + Express.js (Backend)
* MongoDB Atlas
* JWT Authentication
* Recharts

## Prerequisites

* Node.js v20+
* MongoDB Atlas Account

## Environment Variables

Create a `.env` file inside the `backend` folder:

```env
PORT=5000
```

## Backend Setup

```bash
cd backend
npm install
npm run seed
npm run dev
```

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Features

* Downloads Analytics
* DID Analytics
* Uninstall Analytics
* DAU (Daily Active Users)
* WAU (Weekly Active Users)
* MAU (Monthly Active Users)
* JWT Authentication
* MongoDB Atlas Integration
* Real-time Dashboard
